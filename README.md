# Footstep Power Generation System

## Overview
This project demonstrates the generation of electrical energy from footsteps using piezoelectric sensors and Arduino Uno. The generated voltage is monitored on a 16×2 LCD, stored in a rechargeable battery, and used to power a load through a relay module.

## Components Used
- Arduino Uno
- Piezoelectric Sensors
- 16×2 LCD Display
- Relay Module
- Rechargeable Battery
- Bridge Rectifier
- Voltage Regulator
- Connecting Wires

## Working Principle
- Pressure is applied to the piezoelectric sensors.
- The sensors generate electrical energy.
- The generated voltage is rectified and regulated.
- Arduino monitors the voltage.
- The voltage is displayed on the LCD.
- The relay powers the connected load when sufficient voltage is available.
